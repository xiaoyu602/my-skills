# Birthday Skill

管理家人生日，支持阴历和阳历混合存储。
数据保存在 ~/Desktop/my-skills/birthday-skill/birthdays.json。
每周一早上9点自动运行 birthday-check.py，提醒本周过生日的人。

## 数据格式

```json
[
  { "name": "妈妈", "date": "03-15", "type": "solar" },
  { "name": "爸爸", "date": "07-22", "type": "solar" },
  { "name": "奶奶", "date": "12-08", "type": "lunar" }
]
```

- `type` 为 `solar` 表示阳历，`lunar` 表示阴历
- `date` 格式固定为 MM-DD（月-日）
- 阴历日期也用 MM-DD 格式存储，如腊月初八 = 12-08

## 操作指令

### 添加生日

用户说"帮我记录奶奶的阴历生日是腊月初八"：
1. 识别日历类型：有"阴历"/"农历" → type = lunar，否则 → type = solar
2. 把中文日期转成 MM-DD：腊月 = 12月，初八 = 08日 → "12-08"
3. 读取 birthdays.json（不存在则创建空列表）
4. 添加记录并写回文件
5. 回复"✅ 已记录奶奶的生日：阴历 12月08日"

**中文月份对照：**
正月=01 二月=02 三月=03 四月=04 五月=05 六月=06
七月=07 八月=08 九月=09 十月=10 冬月/十一月=11 腊月/十二月=12

**中文日期对照：**
初一=01 初二=02 初三=03 初四=04 初五=05 初六=06 初七=07
初八=08 初九=09 初十=10 十一=11 十二=12 十三=13 十四=14
十五=15 十六=16 十七=17 十八=18 十九=19 二十=20
廿一=21 廿二=22 廿三=23 廿四=24 廿五=25 廿六=26
廿七=27 廿八=28 廿九=29 三十=30

### 查看生日列表

用户说"查看所有生日"或"谁的生日快到了"：
读取 birthdays.json，按月份排序后回复：

```
🎂 生日列表

👤 妈妈 — 阳历 3月15日
👤 奶奶 — 阴历 腊月初八
👤 爸爸 — 阳历 7月22日
```

### 删除生日

用户说"删除奶奶的生日"：
找到对应记录删除，写回文件，回复"✅ 已删除奶奶的生日记录"

## 自动提醒说明

每周一早上11点，birthday-check.py 会自动运行：
- 扫描本周（周一到周日）所有过生日的人
- 合并成一条 Mac 桌面通知推送给你
- 通知格式：「本周过生日：妈妈、爸爸，记得送上祝福！」

手动测试：
```bash
python3 ~/Desktop/my-skills/birthday-skill/birthday-check.py
```

## 注意事项
- 只存月份和日期，不存年份，每年自动循环
- 阴历转阳历由 birthday-check.py 在运行时自动处理
